[컴퓨터그래픽스] 프로그래밍 과제 1: shader를 이용해 정 12면체 그리기 - 2020062324 이은비 

* 정십이면체 구성 *
그레이 / 빨간색 / 노란색 / 연두색 / 파란색 / 마젠타 / 흰색 / 시안 / 핑크 / 보라색 / 갈색 / 올리브색
- 처음 프로그램을 시작했을 때 물체는 정지 상태입니다. 스페이스바를 사용하여 물체의 회전 상태를 바꿀 수 있습니다.

-------------------------------------------------------------
* 조작법 *
q(Q) / ESC : 프로그램 종료.
space bar : 회전 시작/종료.
마우스 왼쪽 클릭 : X축을 기준으로 회전.
마우스 오른쪽 클릭 : Y축을 기준으로 회전.
마우스 중간버튼 클릭 : Z축을 기준으로 회전.

j : 회전 속력 줄이기 (기본 속력에서 최대 4번까지 감소합니다.)
k : 회전 속력 높이기 (기본 속력에서 최대 4번까지 증가합니다.)

z : 스케일 축소 (기본 크기에서 최대 2번까지 축소됩니다.)
x : 스케일 확대 (기본 크기에서 최대 2번까지 확대됩니다.)

a : 물체 왼쪽 이동
d : 물체 오른쪽 이동
w : 물체 위쪽 이동
s : 물체 아래쪽 이동

-------------------------------------------------------------
* 구현상의 특이점 *
- 키보드가 한글 자판으로 설정되어있다면 조작키가 실행되지 않습니다.
- q 를 제외한 나머지 조작키는 소문자, 대문자를 구분합니다.
- 회전하지 않을 때 속력을 올리면, 정지 상태와 상관없이 물체가 회전하는 현상이 있었습니다. 적절한 조건문을 추가하여 해당 현상을 해결하였습니다.
