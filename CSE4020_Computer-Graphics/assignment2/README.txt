[컴퓨터그래픽스 과제2] - 2020062324 이은비
-----------------------------------------------------------------------------
* 구현상의 특이점
- step3 의 Option1 을 선택하여 구현했다 ; 스페이스바로 커브 수정과 화면 움직임을 전환한다. 
- [왼쪽 화면] : 토끼 주변에 커브가 있다.
- [오른쪽 화면] : 토끼를 응시하며 커브를 따라 카메라가 움직인다. 스페이스바를 누르면 움직임이 멈춘다.
-----------------------------------------------------------------------------
* 프로그램 사용법
- 프로그램을 처음 시작하면 '화면 움직임' 모드가 활성화 되어 있다.
- Spacebar 를 누르면 '화면 움직임'은 비활성화 되고, 'curve 수정' 모드로 활성화 된다.
(- 커브 수정 기능이 완성되지 않았다.)
-----------------------------------------------------------------------------
* 버그 사항
- 커브의 control point 가 클릭되었는지(HitIndex) 확인되지 않아, 커브 수정이 불가능하다.
